# Library Management System

A web-based Library Management System built using Spring Boot, Thymeleaf, and MySQL.

## Features
- Manage books (CRUD)
- Manage clients (CRUD)
- Borrow and return books
- Dashboard with statistics
- Search functionality
- User authentication (Admin & Staff roles)
- Role-based access control

## Technologies Used
- Java (Spring Boot)
- Thymeleaf
- MySQL
- Spring Security
- Maven

## Roles
- ADMIN: Full access including dashboard
- STAFF: Access to books, clients, and loans

## How to Run
1. Configure MySQL database
2. Run the Spring Boot application
3. Access: http://localhost:8080

## Login
Use seeded users from the database.
